# 気仙沼シーホークス少年野球クラブ - デザインモック

Next.js 14 App Router で構築したデザインモックサイトです。

## ページ構成

| パス | ページ |
|------|--------|
| `/` | トップページ |
| `/reports` | 活動報告アーカイブ |
| `/reports/[id]` | 活動報告 詳細 |
| `/members` | メンバー紹介 |

## セットアップ

```bash
npm install
npm run dev
```

## Vercel デプロイ

GitHub にプッシュ後、Vercel でインポートすると自動デプロイされます。

## 技術スタック

- Next.js 14 (App Router)
- React 18
- Google Fonts (Noto Sans JP, Bebas Neue)
